🗺️ Documentação de Rotas

Visão geral de todos os endpoints e páginas configurados no sistema de Ninjas.

1. Interface do Usuário (Thymeleaf)

/ninjas/ui/listar

GET

Página: ninjas/listarNinjas
O Quadro de Ninjas.

/ninjas/ui/adicionar

GET

Página: ninjas/adicionarNinja
Formulário de cadastro.

/ninjas/ui/detalhar/{id}

GET

Página: ninja/detalhesNinja
Visualização detalhada de um ninja.

/ninjas/ui/alterar/{id}

GET

Página: ninja/alterarNinja
Formulário de edição.

/ninjas/ui/patch

POST

Ação: Salva a edição e redireciona para a lista.

/ninjas/ui/deletar/{id}

GET

Ação: Exclui o ninja e redireciona para a lista.

2. API REST de Dados (JSON)

/ninjas/criar

POST

Cria um novo ninja e salva no banco (Requer Body JSON).

/ninjas/listar

GET

Retorna o array JSON com todas os ninjas cadastrados.

/ninjas/listar/{id}

GET

Busca os dados de um ninja específico por ID.

/ninjas/alterar/{id}

PATCH

Atualiza campos específicos de um ninja existente.

/ninjas/deletar/{id}

DELETE

Remove um ninja permanentemente do banco.
