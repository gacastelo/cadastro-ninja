🗺️ Documentação de Rotas

Visão geral de todos os endpoints e páginas configurados no sistema de Missões.

1. Interface do Usuário (Thymeleaf)

/missoes/ui/listar

GET

Página: missao/listarMissoes
O Quadro de Missões.

/missoes/ui/adicionar

GET

Página: missao/adicionarMissao
Formulário de cadastro.

/missoes/ui/detalhar/{id}

GET

Página: missao/detalhesMissao
Visualização detalhada de uma missão.

/missoes/ui/alterar/{id}

GET

Página: missao/alterarMissao
Formulário de edição.

/missoes/ui/patch

POST

Ação: Salva a edição e redireciona para a lista.

/missoes/ui/deletar/{id}

GET

Ação: Exclui a missão e redireciona para a lista.

2. API REST de Dados (JSON)

/missoes/criar

POST

Cria uma nova missão e salva no banco (Requer Body JSON).

/missoes/listar

GET

Retorna o array JSON com todas as missões cadastradas.

/missoes/listar/{id}

GET

Busca os dados de uma missão específica por ID.

/missoes/alterar/{id}

PATCH

Atualiza campos específicos de uma missão existente.

/missoes/deletar/{id}

DELETE

Remove uma missão permanentemente do banco.
